BUILTIN

Fonction autorise pour les variables d'environnement : getenv
Fonction non autorise mais qui peuvent etre utiles a coder : setenv, unsetenv (builtin unset), update env. 

PWD : Permet d'afficher le repertoire courant grâce a getcwd.

CD : Permet de changer de repertoire courant grâce a chdir.
	 OLDPWD et PWD doivent être mis à jour dans data->env
	
EXIT: sans argument, bash quitte avec le code de retour du dernier processus éxécuté.
	  avec un argument (exit N), bash quitte avec le code de retour N
	  Si (N >= 0 && N <= 255), le code de retour est N.
	  Si N >= 256, alors le code de retour est N % 256.
	  si N > 0, alors le code de retour est N % 256.

ECHO : doit printf les ARG suivant tant que ce sont des ARG avec un \n à la fin.
	   Si echo -n (ou -nnnnnnnnnnnnnnnnnnnnnnnn), alors echo doit printf les ARG suivants sans 
	   \n à la fin.
	   L'utilisation de echo avec un $ sans quotes ou entre "" doit permettre à echo d'expandre
	   la variable d'environnement -> check si dollar, check quote status et puis check si var existe.
